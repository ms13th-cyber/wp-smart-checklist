投稿の公開前チェックを可視化し、運用ミスを防ぐ軽量なチェックリストプラグインです。

== Description ==

WP Smart Checklist は、投稿やカスタム投稿タイプにチェックリスト機能を追加し、
公開前の確認作業を効率化するためのプラグインです。

チェック項目のカスタマイズ、進捗の可視化、フィルター機能などにより、
チーム運用やクライアント案件でも安心して使用できます。

主な機能：

* 投稿ごとのチェックリスト管理
* チェック進捗のパーセンテージ表示
* 投稿一覧画面への進捗カラム追加
* 完了 / 未完了 / 未チェックでのフィルター
* 状態に応じた色分け表示
* 未完了時の公開アラート表示
* チェック項目のカスタマイズ（設定画面）
* 対象投稿タイプの選択対応

シンプルで軽量ながら、実務で使える機能を備えています。

== Installation ==

1. プラグインファイルを `wp-content/plugins/wp-smart-checklist` にアップロード
2. WordPress管理画面の「プラグイン」から有効化
3. 「設定」→「チェックリスト」から項目を設定

== Usage ==

1. 投稿編集画面の「公開チェックリスト」を確認
2. 各項目にチェックを入れる
3. 保存すると進捗が自動更新されます
4. 投稿一覧でチェック状況を確認できます

== Frequently Asked Questions ==

= チェック項目は変更できますか？ =

はい。「設定」→「チェックリスト」から自由に編集できます。

= カスタム投稿タイプにも使えますか？ =

はい。設定画面で対象の投稿タイプを選択できます。

= 未チェックのまま公開できますか？ =

可能ですが、公開時に確認アラートが表示されます。
