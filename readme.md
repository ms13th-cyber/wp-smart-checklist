# WP Smart Checklist

Optimize your editorial workflow and eliminate "oops" moments. WP Smart Checklist allows you to manage a custom pre-publish task list, ensuring every post meets your quality standards before it goes live.

[日本語の解説は英語の後にあります]

---

## Key Features

- **Customizable Task Lists**: Easily define your own checklist items (e.g., "Check featured image," "SEO description OK?") via the global settings page.
- **Visual Progress Tracking**: Adds a "Check Status" column to the post list, color-coding rows (Red, Orange, Green) based on completion percentage for at-a-glance management.
- **Pre-Publish Guard**: Includes a smart JavaScript alert that warns authors if they attempt to publish a post with unfinished checklist items.
- **Advanced Filtering**: Filter your post list by "Complete," "Incomplete," or "Not Checked" to quickly identify content that needs attention.
- **Multi-Post Type Support**: Select exactly which post types (Posts, Pages, or Custom Post Types) should display the checklist.

## Installation

1. Upload the `wp-smart-checklist` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > チェックリスト (Checklist Settings)** to define your tasks and target post types.

## 主な機能（日本語）

投稿公開前のルーチン作業を確実に管理し、公開ミスを防ぐための強力なチェックリストプラグインです。

- **自由なチェック項目設定**: 「アイキャッチは設定したか？」「SEOメタ記号はOKか？」など、サイト独自のチェックリストを管理画面から簡単に作成できます。
- **一目でわかる進捗管理**: 投稿一覧画面に「チェック状況」カラムを追加。進捗率に応じて行の色が自動で変わる（赤・黄・緑）ため、未完了の記事を即座に特定できます。
- **公開前ガード機能**: チェックが完了していない状態で「公開」ボタンを押すと、確認アラートを表示。ヒューマンエラーをシステム的に防止します。
- **高度なフィルタリング**: 「完了」「未完了」「未チェック」の条件で記事を絞り込み検索でき、編集長や管理者のコンテンツ管理をサポートします。
- **マルチ投稿タイプ対応**: 通常の投稿だけでなく、固定ページやカスタム投稿タイプなど、チェックリストを表示したい対象を個別に選択可能です。

## インストール

1. `wp-smart-checklist` フォルダを `/wp-content/plugins/` にアップロードします。
2. 管理画面の「プラグイン」から有効化してください。
3. 「設定」 > 「チェックリスト」から、項目と対象の投稿タイプを設定できます。
